=== Chicken Disco Dancer ===

HOW TO BUILD:
- Open project root directory.
- mkdir build
- cd build
- cmake -G Xcode ..
- make -j8
- ./FinalProject
- When the xcode project opens, make sure to link a binary library libfmodex.dylib

Technology included:
- Music
- Skinned animations
- Cloth simulation
- Collision detection between skinned character and cloth
